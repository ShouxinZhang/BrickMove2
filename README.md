一个面向 Lean 数学证明的「证明模板」可视化编辑器。

## 快速开始
```bash
bash scripts/start_server.sh
```

## 树形证明工作区（实验版）
- 入口：`frontend/tree/index.html`（运行服务器后打开 `http://localhost:5000/tree/index.html`，或访问旧链接 `http://localhost:5000/tree.html` 自动跳转）。
- 顶栏：导入/粘贴/导出按钮集中在顶部工具栏，可一键打开“树目录预览”浮窗；浮窗内的树状图使用连线展示分支，并可点击节点同步回主界面。
- 节点模型：任意层级的树节点，每个节点包含「符号设定」「子问题描述」「步骤内容（描述 / Math proof / 2 分 / 1 分 API）」，所有步骤以子节点形式呈现，兄弟节点即并列步骤，可随时继续分支。
- 步骤卡片新增 “Math proof” 记事区，用于草稿记录（不会出现在 CSV 导出）。
- 目录支持拖拽排序同级节点（在目录中按住节点行拖拽到目标节点前/后）。
- 导航：左侧树目录保留快速搜索、展开/收起与“根节点子节点”按钮；右侧工作区改为“左侧表单输入 + 右侧实时 Markdown 预览（递归展开全部子节点）”。
- 导入导出：
  1. 支持粘贴/上传新树 JSON；
  2. 兼容旧版 proof JSON / Markdown（自动转换成树结构）；
  3. 导出 JSON / Markdown / CSV，其中 CSV 包含三列：`informal statement`、`score 2 api`、`score 1 api`（第一列拼接符号设定 + 子问题/步骤的自然语言描述）。
- Markdown 预览：实时根据当前节点生成模板化片段，并将子树全部展开；大纲如下：
  ```
  符号设定: …

  问题: …

  step 1. …
  2 score api: …
  1 score api: …
  ```

### 与原编辑器的衔接
- 原本的结构化编辑器（`index.html`）仍然可用；其 JSON/Markdown 导出可以直接在树形工作区导入。
- 在原编辑器中新增的「符号」输入会写入 JSON 的 `symbols` 字段，导入树形工作区后会按层级自动继承并展示。
